# Original JT

**Developer tools, made with care.**

### Source Runner — `sr`

Run self-contained scripts across many runtimes — TypeScript (Node/Deno/Bun),
Python, Ruby, Go, Haskell, Dart, Julia, the shells, and PowerShell — with inline
metadata and shared, cache-isolated dependencies. No `node_modules/` in your
project, no venvs, no global installs: declare your runtime, deps, and tools in a
comment at the top of the file and run it.

```bash
curl -fsSL https://originaljt.com/install.sh | sh   # macOS / Linux
iwr -useb https://originaljt.com/install.ps1 | iex   # Windows
```
